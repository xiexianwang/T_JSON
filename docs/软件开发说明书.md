# 软件开发说明书 — 江苏莱瑟斯监控控制客户端

---

## 1. 项目简介

本项目是江苏莱瑟斯监控设备的 Windows 上位机控制客户端。用户通过本软件在 PC 上远程连接设备，完成以下核心任务：

- **实时查看**：RTSP 拉流解码显示监控视频画面
- **远程操控**：控制云台转动方向、镜头变倍变焦
- **AI 配置**：切换识别/跟踪工作模式，选择算法模型
- **状态监控**：仪表板实时显示镜头倍率、云台角度、GPS 坐标、激光测距、AI 识别结果
- **地理信息**：天地图上叠加设备位置、视场角扇形、目标标记与轨迹

## 2. 需求分析（按模块）

### 2.1 连接管理模块

| 需求 | 说明 |
|------|------|
| TCP 连接/断开 | 用户配置设备 IP 和端口，发起或断开 TCP 长连接 |
| 自动重连 | 网络断开后指数退避自动重连（最多 10 次） |
| 心跳保活 | 10s 周期发送心跳帧维持连接 |
| 指令确认 | 设备对每条指令返回 ACK 状态码（成功/失败） |

### 2.2 云台控制模块

| 需求 | 说明 |
|------|------|
| 8 方向运动 | 上/下/左/右/左上/右上/左下/右下 |
| 运动停止 | 急停当前云台动作 |
| 速度调节 | Pan/Tilt 速度 1–63 可调，持久化保存 |
| 角度设置 | 输入水平角/垂直角，下发转动到指定角度 |
| 角度标定 | 将当前角度标定为零点 |

### 2.3 镜头控制模块

| 需求 | 说明 |
|------|------|
| 双镜头独立控制 | 可见光镜头（VISCA 协议）和红外镜头（Pelco-D 协议） |
| 变倍 Zoom | 放大/缩小，速度 0–7 可调 |
| 变焦 Focus | 远焦/近焦，速度 0–7 可调 |
| 停止镜头 | 停止当前变倍/变焦动作 |

### 2.4 预置位模块

| 需求 | 说明 |
|------|------|
| 调用预置位 | 快速转到预存位置 |
| 设置预置位 | 保存当前位置到编号（1–255） |
| 删除预置位 | 移除指定编号预置位 |

### 2.5 工作模式与 AI 配置模块

| 需求 | 说明 |
|------|------|
| 工作模式切换 | 关闭 AI / 识别 / 自动跟踪 / 点选跟踪 / 框选跟踪 |
| 显示模式切换 | 大图可见光 / 红外全屏 / 可见光全屏 / 融合 / 大图红外 |
| 算法模型选择 | 传感器（可见光/红外）× 识别类型（人车/船/无人机/飞机直升机/鸟） |
| 框选跟踪 | 在视频画面拖拽框选目标，自动进入跟踪模式 |

### 2.6 视频解码模块

| 需求 | 说明 |
|------|------|
| RTSP 拉流 | 连接设备 RTSP 地址拉取实时视频流 |
| H264/H265 解码 | 使用 FFmpeg 硬/软件解码 |
| 实时渲染 | 16:9 自适应显示，QPainter 高效绘制 |
| 断线自动重连 | RTSP 断开后在独立线程中自动重连 |

### 2.7 设备状态仪表板

| 需求 | 说明 |
|------|------|
| 云台状态 | 水平角、垂直角、激光距离、GPS 坐标、高度 |
| 镜头状态 | 可见光倍率/焦距/视场角、红外倍率/焦距/视场角 |
| 识别结果 | 目标 ID/类别/距离/像素坐标/脱靶量表格 |

### 2.8 图像抓拍模块

| 目标轨迹 | 跟踪目标运动轨迹折线绘制 |
| 设备信息悬浮窗 | GPS/高度/距离/方位角/俯仰角/视场角 |

### 2.11 参数配置模块

| 需求 | 说明 |

### 3.3 T-JSON 通信协议

**帧格式**：固定 2 字节帧头（0xEC 0x91）+ 1 字节类型 + 4 字节大端长度 + 变长 JSON/二进制载荷

**协议分层**：

- 传输层：QTcpSocket 异步读写，粘包通过环形缓冲区 + 帧头长度解析
- 帧类型层：17 种 FrameType 区分指令/状态/心跳/ACK/抓拍
- 应用层：JSON ControlType 字段标识具体业务指令（SetWorkMode / ModelSetting 等）
- 串口透传层：SerialControl 封装 Pelco-D / VISCA 底层协议数据包

**关键机制**：

- 心跳 10s 双向保活，4 字节无载荷帧
- 重连指数退避（2s→4s→...→60s），最大 10 次
- WebEngine 远程调试端口 9999，Chrome 可实时调试地图页面
